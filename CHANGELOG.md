# Changelog

All notable changes to the DjangoPlay Documentation repository will be documented in this file.

This repository contains the DjangoPlay documentation portal and engineering documentation for CodeFleet Labs projects.

---

## [1.0.0] - 2026-03-29

### Added

* Initial release of DjangoPlay Documentation repository
* Static documentation portal
* Documentation landing page
* Projects landing page
* Markdown documentation viewer
* Sidebar navigation generated from docs-index.json
* Folder navigation for documentation directories
* Previous and next document navigation
* Full-text search using search-index.json
* Grouped and ranked search results
* Breadcrumb navigation
* Auto-generated Table of Contents from document headings
* Scroll spy for Table of Contents
* Heading anchor links and section deep linking
* Documentation structure for projects, common, governance, templates, and diagrams
* Documentation templates
* Architecture documentation and diagrams
* Deployment documentation
* Runbooks documentation
* Governance and documentation standards
* Theme toggle (dark/light)
* Background animation and documentation portal UI
* Modular JavaScript documentation engine
* Documentation index generation scripts
* Search index generation script

---

This marks the first public release of the DjangoPlay Documentation Portal and Documentation Repository.
